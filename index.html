Vercel الأسرع والأسهل

  1. اذهب إلى vercel.com
  2. سجّل بحساب Google أو GitHub
  3. اضغط Add New → Project
  4. ارفع المجلد كاملاً (بعد فك الضغط)
  5. اضغط Deploy ✅
  6. رابطك: yoursite.vercel.app

Netlify بديل ممتاز

  1. اذهب إلى netlify.com
  2. سجّل بحساب Google
  3. اسحب المجلد وأفلته في الصفحة
  4. انتظر ثوانٍ ✅
  5. رابطك: yoursite.netlify.app
  6. غيّر الاسم من الإعدادات
⚠️ مهم: ارفع المجلد الكامل يحتوي على: index.html + مجلد api/ + vercel.json
الـ Backend (api/download.js) هو من يحل مشكلة CORS ويجلب الفيديو من الخادم مباشرة
